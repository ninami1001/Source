Korišten je ASP .Net Web Aapplication(.NET Framework) sa template Web API RESTful HTTP service. Kreirana je mala baza sa 4 tabele da bi se mogli importovati podaci. Za frontend je korišten Angular u kojem je napravljen import
button koji poziva akciju  na apiju za import podataka.
Prije importa očisti se baza, te je napravljen tabelarni prikaz na frontendu koji prikazuje listu korisnika sa nekoliko podataka i za svakog korisnika prikazani su "Posts" podaci u mat accordionu. Kako se radi o pokaznom primjeru nije implementirano
upravljanje greškama. 
Nejasan je dio traženog prikaza Postova. Naime na prvoj slici je prikazano da se zahtjeva dugme koje će prikazati postove, dok je na drugoj slici prikazano da su u jednom redu korisnički podaci, a drugom redu podaci 
određenog posta, pa samim tim je implementirano da na neki način zadovolji oba prikaza.
